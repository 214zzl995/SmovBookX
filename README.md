# SmovBook

**这是一个集Jav爬虫，Jav整理，Hfs文件服务器功能的软件**
**软件主要用于学习rust**

**功能**

- [x] 本地文件检索
- [x] Javdb爬取爬取内容 名称 演员 详情图片等
- [x] 视频文件窗口
- [x] hfs文件服务器功能
- [x] android客户端软件（半成品）
- [x] 检索悬浮球
- [x] 多线程检索（残废功能，javdb会封IP）
- [x] 文件整理 会归档字幕文件以及种子文件
- [ ] 视频文件分类
- [ ] 自定义文件标记
